# Welcome

This website is a guide for using the `stmdsp` device and its GUI.

## What is 'stmdsp'?

`stmdsp` is a portable digital signal processing (DSP) workstation consisting of the `stmdsp` device and an accompanying GUI.

This device allows for real-world study of DSP by providing students with the ability to write and apply DSP algorithms to real signals in real-time.

DSP algorithms are written in C++, and are compiled and uploaded to the device memory for immediate use. Externals sources or tools may be used for the device's input and output, or the device may be used on its own with the on-board signal generator and GUI signal logging and viewing support.

The GUI acts as a text editor for writing DSP algorithms. It is also used for device interaction and management:
 * Start and stop signal conversions
 * Load or unload DSP algorithms
 * View algorithm disassembly
 * Measure algorithm execution time
 * Draw signals for visual analysis
 * Log conversion results to a .csv file
 * Signal generator management
