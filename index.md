# Welcome

This website is a guide for using the `stmdsp` device and its GUI.

## Device overview

## GUI

The stmdsp device is supported by a computer-side GUI for algorithm design and analysis.

![Sample view of the GUI](https://raw.githubusercontent.com/tcsullivan/stmdsp/gh-pages/gui_main.jpg)

**Features:**
 * Algorithm file editing and management
 * Algorithm compilation (C++) and disassembly viewing
 * Device interactivity:
     * Start and stop signal conversions
     * Load or unload DSP algorithms
     * Measure algorithm execution time
     * Draw signals for visual analysis
     * Log conversion results to a .csv file
     * Signal generator management
